09.10.01 
[!] Версия 1.02

    Крупные изменения

[+] Сделан просмотр мемо полей в формате FoxPro и DBaseIII (нажать F3)
[+] Сделано добавление записи (Shift+F4)
[+] Добавлены возможносты настройки отображения полей
    - измененять ширину столбца Shift+стрелка вправо/Shift+стрелка влево,
    - изменять заголовок столбца (Shift+Enter)
    - скрывать столбцы (Shift+Del) и показывать их снова (Shift+Ins)
    - сохранить настройки во внешнем файле Shift+F9
[+] Появилась возможность пометки записей (Ins)
    + возможность отбора помеченных записей Ctrl+F
    и обратно - Ctrl+A
[+] Поиск теперь
    - может быть регистронезависимым
    - имеет несколько режимов: 
[+] Появился экспорт во внешний файл Ctrl+U 
    в текстовом, dbf или html формате

    Мелочи

[+] Изменены цвета названий полей и помеченных записей
{+] Enter на поле структуры (F2) - переход на это поле
[+] Мемо поля отображаются словом Memo или memo в зависимости от того есть
    в нем что-нибудь или нет
[+] Запрещено изменение мемо полей
[+] При изменении ширины столбца числовые поля сужаются влево
[+] После выполнения плагина не должны вызываться программы ассоциированные
    с данным расширением
[+] Изменена клавиша перехода к строке, теперь это Alt+F8
[+] Изменен формат показа номеров строк, теперь ширина поля расчитывается
    исходя из числа записей
[+] Появилось жалкое подобие help-файла на английском
[+] Можно указать список расширений, файлы с которыми будут
    открываться как dbf файлы
[+] По Ctrl+C суммируются все числовые поля отмеченных записей
[+] При редактировании доступны только те поля, которые в данный
    момент видны на экране
[+] Операция удаления (Del) учитывает наличие помеченных записей
    (если таковые есть, то удаляются все иначе только текущая)
[+] В конфигурации добавилась галочка "Показать разделитель столбцов"
[+] При переходе к строке (Alt+F5) появилась возможность перейти
    к заданному столбцу. Для этого необходимо через запятую
    указать номер столбца.
[+] В момент просмотра можно изменять конфигурацию (F9)
